Carimbador Maluco é um jeito simples de estampar uma frase nos PDFs baixáveis da sua loja Woocommerce

== Description ==

= Em Português Brasil =

Plugin usado para estampar uma frase, que pode usar dados do comprador, nos PDFs baixáveis da sua loja woocommerce.
Fácil de usar. Basta definir o texto e os termos substitutivos na tela de configuração e sempre que alguem baixar um PDF vendido pela sua loja, o arquivo vai ser estampado com o texto  definido, na mergem superior de todas as páginas.

= in English =

Plugin used to print predefined text in the downloadable files of your woocommerce products.
Easy to use. Just set the text in the configuration area, and whenever someone downloads a PDF file sold by your store woocommerce, this will leave with the text stamped in the margin of all pages

== Installation ==

= Instalação =
* Fazer upload para o /wp-content/plugins/ diretório.
* Verifique se já tem instalado o outro Plugin WooCommerce PagSeguro fundamental para a instalação deste.
* Ative o plugin através do menu 'Plugins' no WordPress.

== Minimum Requirements ==

* WordPress 3.8+
* PHP version 5.3+
* Woocommerce 3.0+
